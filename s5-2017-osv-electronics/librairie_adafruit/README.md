#Comment installer une librairie Adafruit qui permet de programmer un baragraph à leds.
#1. Vous devriez copier les deux dossiers sur Adafruit_LEDBackpack et Adafruit-GFX-Library-master.
#2. Aller dans votre programme files, cliquer sur Arduino, ensuite cliquer sur Librairie pour coller les deux dossier.
#3. Redemarer votre environnement de programation Arduino pour voir la bibliothèque Adafruit installé.
